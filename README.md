# deno-starter

简单的 [deno](https://deno.land/) 项目模板

<br />

## Usage

> 注意未加任何权限，请前往 [deno.jsonc](deno.jsonc) 手动添加

### start

```shell
deno task start
```

### release

```shell
deno task release # 将修改 src/version.ts 中的版本号
```
